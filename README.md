# Web Calculator

Calculator application implemented with Reverse Polish notation.

Example expression:

    (2.5-3)*∫(2;[5+7];10;3)-√(80/2+3^2)  

## Building

    mvn package

## Running

    java -jar target/webcalc-1.0.jar

Open in web browser: [http://localhost:8080](http://localhost:8080)
